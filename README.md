## üîß How resumeX Works

resumeX processes resumes in six clear steps to make sense of complex documents:

1. **File Upload:** You add resume files in PDF or DOCX format.
2. **Parsing:** The app reads the file content, recognizing text and structure.
3. **Information Extraction:** It picks out standard data fields like names, contacts, skill tags, job history, and education.
4. **Classification:** Resumes are sorted into 7 different HR types for organization.
5. **Storage:** All extracted data is stored in a local SQLite database on your machine.
6. **Matching:** Given a job description, the app measures how well candidate information fits the job requirements.

This method allows resumeX to handle many resumes quickly while keeping results accurate.
